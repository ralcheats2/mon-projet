# 🔍 Deal Hunter — Chasseur de Bonnes Affaires

Application Windows de détection automatique de bonnes affaires sur LeBonCoin, Vinted et eBay.

## 🚀 Lancement rapide

```bash
# 1. Double-cliquer sur START.bat
# OU manuellement :
pip install -r requirements.txt
python app.py
```

Ouvre ensuite **http://localhost:5000** dans ton navigateur.

## 🎯 Fonctionnalités

- Scraping LeBonCoin, Vinted, eBay en temps réel
- Détection intelligente des objets de valeur (même dans des lots)
- Calcul automatique du score de bonne affaire
- Reconnaissance d'objets même sans mots-clés dans le titre
- Interface web moderne avec filtres
- Alertes visuelles sur les deals <50% du prix normal

## 📦 Prérequis

- Python 3.8+
- Windows 10/11
